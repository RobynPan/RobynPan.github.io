# Solarpunk Jewels

tags: 3D Print, Biomaterial, Solarpunk, Fabrication, Art

Jewellery that imagines a post-scarcity, solar-powered future — grown from bio-materials and printed geometries inspired by ecological systems.

The pieces are designed to feel like artefacts from a world that chose a different path: where manufacturing is distributed, materials are grown not mined, and adornment is political. Each piece embeds a tiny solar cell that trickle-charges a capacitor, powering a single blinking LED — heartbeat slow, barely visible in daylight, obvious in the dark.

## Process

Forms are designed in OpenSCAD using parametric scripts — the same generator can produce a hundred variations by tweaking a handful of numbers. This feels philosophically right: abundant, endlessly variable, never quite identical.

The bio-resin base is a mix of pine rosin and linseed oil, cast in silicone moulds. It's not perfectly durable, but it's compostable, which matters more to me.

## What it means

Solarpunk is less a genre and more a question: what would it look like if we actually tried? These pieces are small wearable answers. Wearing one is a soft act of worldbuilding.
